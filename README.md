  - print out basic information about a loaded dataset (sample, dtypes, NA check, ...),
  - enable easy visual exploration of categorical features,
  - enable easy visual exploration of numerical features using plot type of your choice,
  - summarise statistical information on numerical features and display histograms and pp-plots, to provide context on normality and a fitting correlation coefficient,
  - process data and visualize correlations of numerical features.
## 0. Import dataset from Kaggle and create copy of df_raw
```python
# Install dependencies as needed:
# pip install kagglehub[pandas-datasets]
import kagglehub
from kagglehub import KaggleDatasetAdapter

# Set the path to the file you'd like to load
file_path = "iris_extended.csv"

# Load the latest version
df_raw = kagglehub.load_dataset(
  KaggleDatasetAdapter.PANDAS,
  "samybaladram/iris-dataset-extended",
  file_path
)

# create copy of df_raw
df = df_raw.copy()
```
